Why AgentSON

Every AI coding tool stores your sessions in its own database. There's no export. No standard. No way out. AgentSON fixes that.

You own your data

GDPR Article 20 gives you the right to export your personal data in a machine-readable format. Your AI coding sessions are your data. AgentSON makes that export real.

GDPR Art. 20 · EU Data Act Ch. VI · CCPA

Format, not framework

AgentSON is a JSON schema, not a protocol. No server. No SDK. No lock-in. Just a format that every tool can read and write. Build readers, not integrations.

ADR-008 · vendor neutrality

Trajectory semantics

Quick Start

# Install
pip install agentson

# List sessions from Claude Code
agentson list --tool claude-code

# Export a session
agentson export --tool claude-code --session ses_xxx

# Export to fine-tuning format (Unsloth/ShareGPT)
agentson finetune session.AgentSON --format unsloth --output training.jsonl

# Search across all sessions
agentson search "nightscout setup"

# View in browser (drag & drop)
open docs/viewer/
