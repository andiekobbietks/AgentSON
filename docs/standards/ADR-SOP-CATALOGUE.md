## How to actually navigate this without going crazy

Start anywhere:
- **From a SOP** → its "Governed by" row names the ADRs whose decisions
  shaped it → click through to any of those.
- **From an ADR** → its own "See also" section (bottom of every ADR file)
  links sideways to related ADRs and down to the SOPs it governs — the
  table above is the same information as a flat, scannable index; the
  in-file links are for when you're already reading one and want the
  next hop without leaving the document.
- **The four open ADRs (013, 015–017)** all cross-link each other and
  back to [018](adrs/adr-018-adopt-project-brief-as-scope-charter.md) —
  that's the cluster to read together if you're doing WP1 decision work
  in one sitting.

This catalogue itself isn't an ADR — it's an index. If it goes stale
(new ADR added, no row here), that's a docs bug, flag it same as any
other.
